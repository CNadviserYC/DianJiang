## 📖 简介

点将（DianJiang）是一款专为三国杀桌游玩家设计的武将选择工具。告别传统的纸牌选将，通过手机应用实现更便捷、更公平的武将分配和选择体验。

### 功能介绍

- 🎯 **公平随机** - 使用算法随机分配武将，杜绝作弊
- 📱 **简单易用** - 直观的界面，无需复杂操作
- 🌐 **局域网联机** - 多人游戏自动同步
- 🎮 **双模式支持** - 同时支持国战和军争两种玩法
- 💾 **智能缓存** - 图包自动保存，无需重复上传
- 🎨 **自定义图包** - 支持导入自己的武将图片

---

## ✨ 功能特性

### 🎴 双游戏模式

#### 国战模式
- 自定义每人抽取武将数量
- 从抽到的武将中选择2位
- 横向/纵向展示选择的武将
- 支持多人同时抽取

#### 军争模式
- 随机分配身份牌（主公、反贼、内奸、忠臣）
- 自定义每种身份的数量
- 每人只能选择1位武将
- 同时展示身份牌和武将牌

### 🌐 网络对战功能

- **自动房间发现** - 基于NSD（Network Service Discovery）技术
- **局域网联机** - 同一WiFi下自动组网
- **实时同步** - 所有玩家同步抽取和选择
- **房主控制** - 房主设置游戏参数并开始游戏
- **模式同步** - 玩家加入时自动同步房主的游戏模式

### 🎨 武将展示与交互

- **双击切换** - 双击武将牌切换到替换图片
- **长按旋转** - 长按旋转90度，适配不同视角
- **状态记忆** - 记住每张牌的旋转和替换状态
- **身份牌交互** - 军争模式下身份牌同样支持交互

### 💾 智能存储

- **图包持久化** - 首次上传后自动保存
- **自动加载** - 重启应用自动恢复上次的图包
- **一键更换** - 随时可以切换新的图包
- **缓存管理** - 智能管理存储空间

## 🐛 常见问题

### Q1: 上传图包后提示"未找到武将图片"

**原因**: 图片文件命名不符合要求

**解决方案**:
- 确保图片文件名是纯数字（如：1.jpg, 2.png）
- 不要包含汉字、字母或特殊字符
- 检查文件扩展名是否正确

### Q2: 找不到房间或无法加入

**原因**: 网络配置问题

**解决方案**:
- 确认所有设备连接同一WiFi
- 检查WiFi是否启用了AP隔离
- 尝试关闭并重新打开WiFi
- 重启应用
